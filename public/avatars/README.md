# 自定义图片头像

把图片文件（`.png` / `.jpg` / `.jpeg` / `.gif` / `.webp` / `.svg`）放进这个文件夹，
启动服务器后它们会自动出现在「选择头像」里——无需改任何代码。

## 自动规整（无需手动处理）

App 读取这些图片时会自动处理，所以**对原图尺寸/比例没有硬性要求**：

- **不是正方形** → 自动从中心裁出最大正方形（居中，不会被拉伸变形）。
- **太大** → 自动缩小到 256×256 以内，并用 WebP 有损压缩降低画质/体积。
- 小图保持原尺寸，不会被放大（过小的图会偏糊，建议 ≥128px）。

也就是说：直接丢原图进来即可，剩下的交给程序。

格式建议 PNG / WebP；文件名用英文/数字更稳妥（避免个别浏览器对中文路径的兼容问题）。

## 版权提醒

只放你**自己拥有版权或已获授权**的图片。宝可梦、暴走漫画、电影截图
（如《赌神》《赌圣》中的演员剧照）等都受版权/商标保护，请勿放入并分发，
以免侵权。需要这类风格时，建议使用你自己有权使用的素材或自行绘制的原创头像。
